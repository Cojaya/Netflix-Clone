A fully responsive Netflix landing page clone built using HTML, CSS, and basic JavaScript, closely replicating the official Netflix India homepage.
Designed with pixel-perfect UI, animations, layout precision, and responsiveness—ideal for showcasing frontend skills.

This project recreates the hero section, TV/video animations, download section, FAQ-style layout, and footer elements of Netflix.

# Project Overview
This Netflix Clone aims to closely mirror the original Netflix homepage, including:
- A powerful hero section
- Overlay-based UI
- Video-inside-TV animations
- Visually attractive content sections
- A completely responsive layout
- A clean and consistent modern theme
  
The entire project is built without using any frameworks, showcasing pure frontend design skills.

# Technologies Used

| Technology                               | Purpose / Why It Was Used                                                          |
| ---------------------------------------- | ---------------------------------------------------------------------------------- |
| **HTML5**                                | Creates the structure of the webpage, organizes content, sections, and layout.     |
| **CSS3**                                 | Adds styling, visuals, responsiveness, animations, and enhances user experience.   |
| **Flexbox**                              | Aligns items neatly across sections, ensures clean spacing and responsive layouts. |
| **Google Fonts (Poppins & Martel Sans)** | Provides a premium Netflix-style typography for headings and body text.            |
| **Images & Background Assets**           | Used for hero background, mockups, device frames, and UI elements.                 |
| **Embedded Videos**                      | Creates TV-frame animation effect similar to Netflix homepage.                     |

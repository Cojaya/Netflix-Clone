A fully responsive Netflix landing page clone built using HTML, CSS, and basic JavaScript, closely replicating the official Netflix India homepage.
Designed with pixel-perfect UI, animations, layout precision, and responsiveness—ideal for showcasing frontend skills.

This project recreates the hero section, TV/video animations, download section, FAQ-style layout, and footer elements of Netflix.

# Project Overview
This Netflix Clone aims to closely mirror the original Netflix homepage, including:
- A powerful hero section
- Overlay-based UI
- Video-inside-TV animations
- Visually attractive content sections
- A completely responsive layout
- A clean and consistent modern theme
  
The entire project is built without using any frameworks, showcasing pure frontend design skills.

### 2. Enjoy On TV – TV + Video Animation
This section highlights how Netflix plays videos seamlessly across devices.

It contains:
- A TV frame image
- A video that auto-plays inside the TV screen

This creates the illusion that the TV itself is playing the video — just like Netflix does on their homepage.

It demonstrates:
- Layering multiple elements
- Combining images + video
- Creating a realistic animation effect using only CSS position management

### 3. Download  Shows – Mobile + Offline Feature
This block focuses on Netflix’s offline download feature.

It displays:
- A mobile device mockup
- Descriptive text explaining how users can save shows

This section is designed to communicate:
- Mobility
- Convenience
- Real-world usage

It also introduces variety in layout by showing image on one side and text on the other, giving a natural browsing flow.

### 4. Watch Everywhere – Multi-Device Streaming

It includes:

#### "Entertainment anytime, anywhere."
